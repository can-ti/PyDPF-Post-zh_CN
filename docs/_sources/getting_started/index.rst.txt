===============
Getting started
===============

要使用 PyDPF-Post API，必须安装 PyDPF-Core。由于需要使用本地 DPF 服务器，您还必须在本地安装 Ansys 2021 R1 或更高版本或独立的 DPF 服务器。
有关获取 Ansys 授权副本的详细信息，请访问 `Ansys 网站 <https://www.ansys.com/>`_ 。
有关获取独立 DPF 服务器的更多信息，请访问 `PyDPF-Core 文档 <https://dpf.docs.pyansys.com/dev/user_guide/getting_started_with_dpf_server.html>`_ 。

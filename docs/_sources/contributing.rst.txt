.. _contributing:

==========
Contribute
==========

在 *PyAnsys 开发者指南* 的 `Contribute <https://dev.docs.pyansys.com/overview/contributing.html>`_ 中，
有关于向 PyAnsys 代码库贡献的总体指导。在尝试向 PyDPF-Post 投稿之前，请确保您已经完全熟悉了这份指南，
尤其要注意 `Guidelines and Best Practices <https://dev.docs.pyansys.com/guidelines/index.html>`_ 。
 
以下是 PyDPF-Post 特有的贡献信息。

Clone the repository
--------------------

要在开发模式下克隆并安装最新版本的 PyDPF-Core，请运行:

.. include:: pydpf-post_clone_install.rst


Post issues
-----------

使用 `PyDPF-Post Issues <https://github.com/ansys/pydpf-post/issues>`_ 页面提交问题、报告错误和申请新功能。

如需联系项目支持团队，请发送电子邮件至 `pyansys.core@ansys.com <pyansys.core@ansys.com>`_ 。

View documentation
------------------
PyDPF-Post 最新稳定版的文档托管在 `PyDPF Documentation <https://post.docs.pyansys.com/>`_ 。

在文档标题栏的右上角有一个选项，可以从查看最新稳定版本的文档切换到查看开发版本或以前发布版本的文档。
